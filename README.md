# Pokemon-API-Training
A simple web application that I'm creating to experiment with JavaScript DOM manipulation and API integration. 
To do this, I'm going to create an interactive Pokedex that fetches Pokémon data from the PokeAPI and displays it dynamically.

## Project Overview
Within this project, I'll explore:
- JavaScript DOM manipulation
- API integration and data fetching
- Dynamic content rendering
- User input handling

## Features

**Currently Implemented:**
- User input capture and display
- Display Pokémon sprite/image

**In Progress:**

**Planned:**
- Dynamic spirits (interactable?)
- Search Enhancements (search by ID, autocomplete)
- List view (e.g. like Pokémon website)
- Pokemon details expansion (abilities, stats, moves)
- Favorites System
- Evolution Chain
- Routing / Multi-Page


Technologies Used
- Frontend: HTML, JavaScript
- API: [PokeAPI]([url](https://pokeapi.co/)) - RESTful Pokemon API
- Hosting: **Currently** LocalHost
